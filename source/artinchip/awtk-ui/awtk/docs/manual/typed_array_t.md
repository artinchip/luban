## typed\_array\_t
### 概述
带类型的动态数组，根据元素个数动态调整数组的容量。

示例：

```c
value_t v;
typed_array_t* typed_array = typed_array_create(VALUE_TYPE_INT, 10);
typed_array_push(a, value_set_int(&v, 123));
typed_array_push(a, value_set_int(&v, 234));
...
typed_array_destroy(typed_array);
```
