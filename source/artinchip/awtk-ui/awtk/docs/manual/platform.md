#### get\_time\_ms64 函数
-----------------------

* 函数功能：

> <p id="platform_get_time_ms64"> 获取当前时间。



* 函数原型：

```
uint64_t get_time_ms64 ();
```

* 参数说明：

| 参数 | 类型 | 说明 |
| -------- | ----- | --------- |
| 返回值 | uint64\_t | 成功返回当前时间。 |

* 函数原型：


| 参数 | 类型 | 说明 |

* 参数说明：

| 参数 | 类型 | 说明 |
