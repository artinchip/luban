## wstr\_t
### 概述
可变长度的宽字符字符串。

示例：

```c
wstr_t s;
wstr_init(&s, 0);

wstr_append(&s, L"abc");
wstr_append(&s, L"123");

wstr_reset(&s);
```
> 先调wstr\_init进行初始化，最后调用wstr\_reset释放内存。

#### wstr\_attach 函数
-----------------------

* 函数功能：

> <p id="wstr_t_wstr_attach">通过附加到一个buff来初始化str。
>可以避免str动态分配内存，同时也不会自动扩展内存，使用完成后无需调用str_reset。

```c
wstr_t s;
wchar_t buff[32];
wstr_attach(&s, buff, ARRAY_SIZE(buff));
wstr_set(&s, L"abc");
wstr_append(&s, L"123");
```

* 函数原型：

```
wstr_t* wstr_attach (wstr_t* str, wchar_t* buff, uint32_t capacity);
```

* 参数说明：

| 参数 | 类型 | 说明 |
| -------- | ----- | --------- |
| 返回值 | wstr\_t* | str对象本身。 |
| str | wstr\_t* | str对象。 |
| buff | wchar\_t* | 缓冲区。 |
| capacity | uint32\_t | 初始容量。 |
