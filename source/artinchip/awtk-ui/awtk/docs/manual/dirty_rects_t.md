## dirty\_rects\_t
### 概述
支持多个脏矩形。
> 在通常情况下，脏矩形的个数并不多，而且一般都是不重叠的，
> 所以为了降低计算开销、避免内存分配和简化实现。这里采用下列措施：

* 如果新的脏矩形是独立的，直接加入进来。
* 如果新的脏矩形与某个脏矩形有重叠，则合并到该脏矩形。
* 如果脏矩形的个数超出最大个数则进行合并。
