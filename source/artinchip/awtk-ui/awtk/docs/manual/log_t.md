#### log\_debug 函数
-----------------------

* 函数功能：

> <p id="log_t_log_debug">输出调试信息。
> 变参函数。

```
log_debug("debug message\n");
```

* 函数原型：

```
void log_debug (const char* format);
```

* 参数说明：

| 参数 | 类型 | 说明 |
| -------- | ----- | --------- |
| 返回值 | void | 无返回值。 |
| format | const char* | 格式或信息。 |
