#### path\_abs 函数
-----------------------

* 函数功能：

> <p id="path_path_abs">返回绝对路径。


* 函数原型：

```
ret_t path_abs (const char* path* path, char* result, uint32_t size);
```

* 参数说明：

| 参数 | 类型 | 说明 |
| -------- | ----- | --------- |
| 返回值 | ret\_t | 返回RET\_OK表示成功，否则表示失败。 |
| path | const char* path* | 路径。 |
| result | char* | 用于返回绝对路径。 |
| size | uint32\_t | 缓冲区大小。 |
