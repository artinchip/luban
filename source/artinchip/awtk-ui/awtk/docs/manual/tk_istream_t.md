#### tk\_istream\_eos 函数
-----------------------

* 函数功能：

> <p id="tk_istream_t_tk_istream_eos">判断数据是否读完。

* 函数原型：

```
bool_t tk_istream_eos (tk_istream_t* stream);
```

* 参数说明：

| 参数 | 类型 | 说明 |
| -------- | ----- | --------- |
| 返回值 | bool\_t | 返回TRUE表示数据读完，否则表示还有数据。 |
| stream | tk\_istream\_t* | istream对象。 |
