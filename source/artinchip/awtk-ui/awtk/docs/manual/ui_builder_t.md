## ui\_builder\_t
### 概述
界面构造器接口。

负责把ui\_loader解析的数据进一步处理，至于具体做什么，由具体实现决定。常见的实现有：

* 构建widget对象树。

* 生成二进制的UI描述数据。

* 生成XML格式的UI描述数据。
