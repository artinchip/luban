eXpanded Serial Peripheral Interface (xSPI)

15 Jan 2025
Read time: 1 minute(s)

xSPI 模块是一个高度灵活和可配置的接口，能够支持多种不同的串行和并行总线协议，包括外部并行总线 Xccela、HyperBus、OPI、QPI、和1/ 2/ 4/ 8 线 SPI。xSPI 模块可与外部 PSRAM、Flash 存储设备等元器件实现连接，无需额外的硬件接口或适配器。

功能特性

  • 支持 Xccela 协议
  • 支持 HyperBus 协议
  • 支持 OPI 协议
  • 支持 QPI 协议
  • 支持 1/ 2/ 4/ 8 线 SPI
  • 支持 200 MHz DDR/ SDR 采样
  • 支持 XIP (eXecute In Place)
  • 支持中断以及 DMA
  • 支持两片并行读写数据
  • 支持单片 16 bits 位宽读写数据
  • 具有 16x32 bits TXFIFO 以及 16x32 bits RXFIFO
  • 具有 512 MB 内存映射空间
