删除开发板

15 Jan 2024
Read time: 1 minute(s)

Luban SDK 提供快速删除开发板配置的方法。

用户可在选择一个需要删除的配置文件后，使用 del_board 程序删除和这个配置文件相关的目录和配置，详细流程如下：

  1. 在 SDK 根目录下执行 make del_board 命令。

  2. 根据提示，提供必要的信息。例如：

    Reference defconfig:(Delete one board base on selected defconfig)
    d211_demo100_nand_defconfig
    d211_demo128_nand_defconfig
    d211_demo88_nand_defconfig
    d211_demo88_nor_defconfig
    d211_demo88_nor_musl_defconfig
    d211_demo_defconfig
    d211_ota_defconfig
    d211_ota_emmc_defconfig
Select reference defconfig for detele board(number): 1
        d211_demo100_nand_defconfig

  3. 在创建完相关的目录和配置之后，del_board 程序会列出删除的目录和相关配置文件。

    Deleted: target/d211/demo100_nand
deleted: source/uboot-2021.10/configs/d211_demo100_nand_defconfig
Deleted: source/linux-5.10/arch/riscv/configs/
        d211_demo100_nand_defconfig
Deleted: package/third-party/busybox/configs/riscv64_linux_defconfig
Deleted: target/configs/d211_demo100_nand_defconfig
Updated: target/d211/Config.in
