Display Engine

15 Jan 2024
Read time: 1 minute(s) 
&de0 {
        status = "okay";

        port@0 {
                reg = <0>;
                de0_in: endpoint {
                        remote-endpoint = <&fb0_out>;
                };
        };

        port@1 {
                reg = <1>;
                de0_out: endpoint {
                        remote-endpoint = <&lvds0_in>;
                };
        };
};
表 1. MIPI-DBI Display Interface 参数描述
参数名称 描述
port@0 数据输入端口，连接 display engine 结点
port@1 数据输出端口，连接到 interface 结点。上述代码示例，表示连接到 lvds 结点。
tearing-effect tearing effetc 模式，支持 bypass / hold / auto 三种模式。三种模式的宏定义取值，可查看tearing effetc 模式宏定义取值。
te-pulse-width TE 信号的脉冲宽度，设置其脉冲宽度等于多少个 pixel clock cycle
表 2. tearing effetc 模式宏定义取值
tearing effect mode macro value 备注
bypass TE_BYPASS 0 连续模式，持续刷新，不启用 te
hold TE_HOLD 1 单帧模式，需要手动刷新
auto TE_AUTO 2 自动单帧模式， 接受到 te 信号后刷新一帧数据
