模块架构

由于使用 SPI 的外设较多，内核中通过 SPI 子系统来支持各种 SPI 外设，整体架构如上图所示，包括：

  • ArtInChip SPI 驱动

  • SPI Core

  • SPI-MEM

  • 面向内核的 API

  • 面向用户空间的接口（SPIDEV）

由于 SPI 传输需要使用 DMA，因此 DMA 子系统是一个相关模块。
